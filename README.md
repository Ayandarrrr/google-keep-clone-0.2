# Google Keep clone

A React + Tailwind notes app with light/dark mode and a collapsible sidebar.

## Setup

```bash
npm install
npm run dev
```

Then open the local URL Vite prints (usually http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```
